# Libraries required

- `TFT_eSPI` by Bodmer.
- `TFT_eWidget` by Bodmer.


# Structure

- `ESP32.ino`, main Arduino sketch.
- `TFTScreen.ino`, handles anything related to the TFT screen, to get touch input from the user.
- `Free_Fonts.h`, provided by the `TFT_eSPI` library.
